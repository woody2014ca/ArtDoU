# 已有学生家长纳入小程序 — 详细指引

## 为什么家长直接扫码会变成「审核员界面」？

小程序通过**当前微信的身份**判断显示什么界面：
- 在「老师表」里 → 教师端
- 在「家长绑定表」里且对应学员已开放家长 → **家长端**
- 都不在 → **审核员界面**（演示用）

**已有学生的家长**还没有被写入「家长绑定表」，所以直接扫码或搜索打开时会被当成「陌生人」，只能看到审核员界面，看不到孩子的课时、作品等。

因此需要先**把家长纳入**（完成绑定），之后他们再扫码或打开小程序，就会自动进入家长端。

---

## 重要：不要先让家长直接扫码

在完成下面任一种「纳入」方式之前，**不要让家长直接扫小程序的普通码**，否则他们会看到审核员界面且没有「我是家长，绑定手机号」入口。

正确做法二选一：
1. **方式一**：老师先在后台用家长的 OpenID 设为该学员家长，**再**让家长扫码。
2. **方式二**：老师先发「家长入口」给家长（见下），家长从**该入口**打开并完成手机号绑定，**之后**家长再扫码就是家长端。

---

## 方式一：老师用 OpenID 直接设为家长（推荐，一步到位）

**适用**：学员档案里已有该家长手机号，且能联系到家长配合发 OpenID。

### 步骤

1. **老师发「家长入口」给家长（仅此一次，用于家长打开并复制 OpenID）**
   - 老师打开小程序，在**首页**点击右上角「**…**」→「**转发**」或「**发送给朋友**」；
   - 选择要纳入的那位家长，发送。
   - 小程序会以「**家长入口**」形式分享（路径带 `from=parent`），家长点开后会看到「意向学员缴费」「我是家长，绑定手机号」。
   - **或**：老师在微信里把小程序**转发给家长**（转发出去的卡片即带家长入口）。

2. **家长操作（仅此一次）**
   - 家长在微信里点开老师发来的**小程序卡片**（不要自己搜小程序或扫普通码）；
   - 进入后应看到「意向学员缴费」「我是家长，绑定手机号」；
   - 点击「**我是家长，绑定手机号**」；
   - 在页面下方找到「**您的 OpenID**」→ 点击「**复制**」；
   - 把复制的内容发给老师（可微信文字发给老师）。

3. **老师操作**
   - 老师打开小程序 → **学员名录** → 找到该学员 → **管理 / Edit**（编辑学员）；
   - 在「**家长 OpenID（指定该学员的家长微信）**」一栏**粘贴**家长发来的 OpenID；
   - 点击「**设为该学员家长**」；
   - 提示成功即可。

4. **之后**
   - 该家长**再打开小程序**（扫码、搜索、从聊天里点均可），都会直接进入**家长端**，看到该学员的课时、作品、请假等，无需再绑定。

**优点**：绑定后家长以后怎么进都是家长端，无需再区分「从哪进」。  
**注意**：家长必须至少通过一次「老师转发的小程序卡片」进入，才能看到绑定页并复制 OpenID；不要让家长第一次就自己扫码。

---

## 方式二：家长用手机号绑定（需先发家长入口）

**适用**：学员档案里已填写家长电话，且已对该学员「开放家长手机号绑定」。

### 步骤

1. **老师在小程序里为学员「开放家长手机号绑定」**
   - 老师打开小程序 → **学员名录** → 找到该学员 → **管理 / Edit**；
   - 确认「**家长电话**」无误（家长将用此手机号绑定）；
   - 点击「**开放家长手机号绑定**」→ 确认；
   - 提示「已开放」即可。

2. **老师发「家长入口」给家长**
   - 老师在小程序**首页**点击右上角「**…**」→「**转发**」或「**发送给朋友**」；
   - 选择该家长并发送（转发出去的卡片带 `from=parent`，家长点开会看到家长入口）。
   - **重要**：必须让家长通过**这次转发的小程序卡片**进入，不要让家长自己搜小程序或扫普通码。

3. **家长操作**
   - 家长在微信里点开老师发来的**小程序卡片**；
   - 应看到「意向学员缴费」「我是家长，绑定手机号」；
   - 点击「**我是家长，绑定手机号**」；
   - 输入**老师登记在该学员名下的家长手机号**（与步骤 1 中一致）；
   - 点击「**确认绑定**」；
   - 绑定成功后会跳转到家长首页。

4. **之后**
   - 该家长再打开小程序（扫码、搜索、从聊天点开均可），都会直接进入**家长端**。

**优点**：家长不用复制 OpenID，只需输入手机号。  
**注意**：学员必须已「开放家长手机号绑定」，且家长**第一次**必须从老师转发的卡片进入，否则看不到「我是家长，绑定手机号」。

---

## 如何批量纳入多名家长？

- **方式一（OpenID）**：对每位家长重复「老师转发家长入口 → 家长复制 OpenID 发给老师 → 老师在对应学员编辑页粘贴 OpenID 并设为该学员家长」。
- **方式二（手机号）**：对每位学员先「开放家长手机号绑定」，再给对应家长**单独转发一次**小程序卡片（或发同一条转发给多名家长，但每人要点一次卡片进入），让家长从卡片进入后点「我是家长，绑定手机号」并输入手机号。

可统一发一段说明给家长，例如：

> 「请点开我发给您的小程序卡片（不要自己搜索小程序），进入后点击「我是家长，绑定手机号」，用您在我这边登记的手机号完成绑定。绑定成功后，您以后打开小程序就能直接看到孩子的课时和作品。」

---

## 老师如何获得「家长入口」链接 / 卡片？

- **转发卡片（推荐）**  
  老师在小程序**首页**点右上角「**…**」→「**转发**」/「**发送给朋友**」，选择家长发送。  
  系统已配置：转发出去的路径带 `from=parent`，家长点开即进入家长入口（能看到「我是家长，绑定手机号」）。

- **小程序码（可选）**  
  如需做成海报或长期链接，可在微信公众平台 → 小程序 → **设置** → **基本设置** 中生成「小程序码」，路径填：`pages/index/index?from=parent`。家长扫该码进入等同从家长入口进入。

---

## 小结

| 步骤 | 方式一（OpenID） | 方式二（手机号） |
|------|------------------|------------------|
| 1 | 老师转发首页给家长 | 老师为学员「开放家长手机号绑定」 |
| 2 | 家长点卡片进入 → 点「我是家长，绑定手机号」→ 复制 OpenID 发给老师 | 老师转发首页给家长 |
| 3 | 老师在编辑学员中粘贴 OpenID → 「设为该学员家长」 | 家长点卡片进入 → 点「我是家长，绑定手机号」→ 输入手机号绑定 |
| 4 | 家长再打开小程序即为家长端 | 同左 |

**共同要点**：  
- 不要让家长**第一次**就自己扫码或搜索小程序；  
- 必须让家长通过**老师转发的带家长入口的小程序卡片**（或带 `from=parent` 的小程序码）进入，才能看到并完成绑定；  
- 完成绑定后，家长再扫码或任意方式打开，都会是家长端。
